📚 Data Guide & How to Play

Complete guide to using the Medical Patient Simulator

🎯 How to Play the Game

Objective: Practice clinical reasoning by taking patient history, asking appropriate questions, and making diagnoses.

📋 Step-by-Step Process:

  1. Generate Patient: Click one of the main buttons to create a random patient case
  2. Review Patient Info: Read the patient's basic information and chief complaint
  3. Start Consultation: Begin asking questions to gather medical history
  4. Ask Follow-up Questions: Use the patient's responses to guide your next questions
  5. Generate MCQs: Create quiz questions to test your understanding
  6. Submit Diagnosis: Make your final diagnosis based on gathered information
  7. Get Feedback: Receive AI-generated feedback on your approach

🔧 Button Functions & Features

💬 Text Chat Only:

  • Pure text interface - No visual distractions
  • Focus on communication and clinical reasoning
  • Best for: Medical students, clinical practice, exam preparation

🎭 3D Avatar Chat:

  • 3D character visualization with realistic proportions
  • Procedural generation - Each patient looks unique
  • Interactive animations based on conversation context
  • Best for: Immersive training, patient interaction practice

🧪 Test Buttons (Small Corner Buttons):

  • Test 2D: 2D animated character interface (experimental)
  • Test 3D: Advanced 3D character with enhanced features (experimental)
  • Purpose: Testing new features and experimental interfaces

🎮 Chat Interface Controls

📝 Text Input:

  • Type your questions in natural language
  • Ask about: Symptoms, medical history, family history, social factors
  • Examples: "What brings you in today?", "How long have you had this pain?", "Any family history of heart disease?"

🔘 Action Buttons:

  • Generate MCQ: Create quiz questions based on current case
  • Submit Diagnosis: Enter your final diagnosis
  • Get Hint: Receive AI-generated guidance
  • New Patient: Start with a different case
  • Reset Chat: Clear conversation history

📊 Data Sources & AI Models

🏥 Medical Database:

  • NHS symptom database - Realistic medical conditions
  • 15+ disease categories with detailed symptom profiles
  • Evidence-based medicine from reliable sources

🤖 AI Technology:

  • OpenRouter API - Access to multiple AI models
  • Mistral, Llama, Qwen - Advanced language models
  • Context-aware responses based on patient personality
  • Consistent character behavior throughout conversation

🎭 Personality System:

  • 5 personality types: Anxious, Confident, Reserved, Talkative, Neutral
  • Communication styles: Formal, Casual, Technical
  • Emotional responses based on symptoms and personality
  • Realistic patient behavior that maintains consistency

💡 Pro Tips for Success

  • Start broad: Ask open-ended questions first
  • Follow the patient's lead: Use their responses to guide follow-up questions
  • Think systematically: Cover all aspects of medical history
  • Use the MCQ feature: Test your understanding as you go
  • Practice regularly: Each case is different and offers new learning opportunities

🔍 Troubleshooting

  • If chat seems stuck: Try refreshing the page or starting a new patient
  • If 3D doesn't load: Ensure your browser supports WebGL and try a different browser
  • For mobile users: Rotate to landscape mode for better experience
  • Performance issues: Close other browser tabs and applications
